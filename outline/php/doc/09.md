# PHP8.1编译配置命令

php8.1编译配置参考命令

```shell
./configure --prefix=/usr/local/php-8.1.3 \
--with-config-file-path=/usr/local/php-8.1.3/etc \
--with-zlib \
--with-zip \
--with-pdo-mysql=mysqlnd \
--with-mysqli=mysqlnd \
--with-mysqli=mysqlnd \
--enable-gd \
--with-external-gd \
--with-jpeg \
--with-xpm \
--with-webp \
--with-freetype \
--with-zlib-dir \
--with-bz2 \
--with-openssl \
--with-curl \
--enable-soap \
--enable-mbstring \
--enable-sockets \
--enable-exif
```
