# PHP7.4编译配置命令

> 版本：PHP 7.4.13

编译配置命令

```shell
./configure --prefix=/usr/local/php \
--with-config-file-path=/usr/local/php/etc \
--with-zlib \
--with-zip \
--with-pdo-mysql=mysqlnd \
--with-mysqli=mysqlnd \
--with-mysqli=mysqlnd \
--enable-gd \
--with-external-gd \
--with-jpeg \
--with-xpm \
--with-webp \
--with-freetype \
--with-iconv-dir \
--with-zlib-dir \
--with-bz2 \
--with-openssl \
--with-curl \
--enable-soap \
--enable-mbstring \
--enable-sockets \
--enable-exif \
--disable-ipv6
```
